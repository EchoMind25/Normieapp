Render grid: stickers.map(sticker => <img src={sticker.src} alt={sticker.name} onClick={() => addToCanvas(sticker.src)} />)
addToCanvas function: fabric.Image.fromURL(src, (img) => { img.set({ left: 100, top: 100, scaleX: 1.5, scaleY: 1.5, opacity: 1, shadow: '0 0 5px rgba(0,0,0,0.5)', borderColor: '#fff', cornerColor: '#fff' }); canvas.add(img); });

For prominent display: Always set opacity: 1 and add shadow/border for visibility on any background. Test on dark/light/user-uploads.
For default background: On load, if no background, set canvas.backgroundColor = '#f0f0f0'; add fabric.Text('Drag stickers or upload background to start your Normie meme!', { left: canvas.width / 2, top: canvas.height / 2, fontSize: 20, fill: '#333', originX: 'center', originY: 'center' }); Remove on first interaction.
