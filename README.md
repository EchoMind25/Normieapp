## Features

- Real-time Solana memecoin price tracking
- Token supply monitoring (burned & locked tokens)
- Interactive price charts with dev buy markers
- NFT collection monitoring
- Community meme generator
- Real-time community chat
- User profiles with wallet integration
- Friend system with E2E encrypted messaging
- Community polls and activity feed
- Whale buy alerts and jeet sell alarms
- Push notifications
- iOS & Android native apps (via Capacitor)

---

## Technology Stack

- **Frontend:** React + TypeScript
- **Mobile:** Capacitor (iOS/Android)
- **Backend:** Node.js + Express
- **Database:** PostgreSQL with Drizzle ORM
- **Real-time Data:** WebSocket integration
- **Blockchain:** Solana via @solana/web3.js
- **Infrastructure:** Echo Mind's automation framework

## Getting Started

### Development

```bash
npm install
npm run dev
```

### Database Setup

```bash
npm run db:push
```

### iOS Build

```bash
npm run build
npx cap sync ios
```
