## Full Description

Normie Observer is your real-time companion app for the $NORMIE memecoin on Solana.

### Features:
- Live price tracking with real-time updates
- Market cap, volume, and holder metrics
- Token supply tracking (burned & locked tokens)
- Interactive price charts with dev buy markers
- Community meme generator with templates
- Community polls and activity feed
- Social media integration
- Whale buy alerts and jeet sell alarms
- Push notifications for major events

### Community Features:
- Real-time community chat
- User profiles with wallet linking
- Friend system with private messaging
- End-to-end encrypted conversations
- Gallery for community-created memes
